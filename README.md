# 🎮 Gamer Mega Pack

A gaming-themed performance dashboard UI built with **HTML, CSS, and JavaScript**.

It simulates an FPS booster overlay, NVIDIA-style tweak panel, and system optimizer tools in a clean gaming interface.

---

## ⚡ Features

- 🎯 Real-time FPS overlay 
- 🚀 FPS booster toggle system
- 🎮 NVIDIA-style tweak panel 
- 🧠 System optimizer buttons
- 🖥️ Gaming dashboard layout


---

## 🧩 What This Project Does


- Modify GPU settings
- Change system performance
- Access hardware or drivers

It is built for:
- UI/UX practice
- Gaming dashboard concepts


---

## 🖥️ How to Run

1. Download or clone the repository
2. Open `index.html`
3. Enjoy the dashboard in your browser

No installation required.
